Hakemistossa nyt esim package.json ja app.js. Hakemiston voi avata vaikka 
Bracketsilla. Tiedosto -> Avaa kansio...

Asentaa dependecyt (tiedostot ja modulit):
	komentokehoitteessa:
	npm install

Serverin starnpm npm ttaus
	npm start
Nyt localhost:3000 näkyy Express title ja teksti. Tuo komento lukee package.json tiedoston, joka
ikäänkuin konffitiedosto. Tai nodemon ./bin/www. Tai muuta package.json filestä node nodemoniksi.

Tyylit: style.styl ylikirjoittaa tyylin aina, kun käynnistää serverin. Tee muutoksen 
siihen, ei css-tiedostoon.

Muutettiin index.js ja index.jade tiedostoja. Käynnistä serveri uudelleen
(npm start), kun .js tiedostoa muutettu.

layout.jade tiedostoon lisättiin jquery mobilen theme koneen linkit.

MongoDB
Lataa tuolta: http://www.mongodb.org/downloads
Asenna
Luo folderti c:/data
ja c:/data/db
MongoDB käyttää niitä defaulttina.

Testaa, toimiiko:
Mene tonne: C:\Program Files\MongoDB 2.6 Standard
Avaan biniin komentokehote (shift+oikea näppäin...)
Komentokehotteessa: mongod (sallitteko käytön, joo).
Aina kun sulkee koneen, mongodb sulkee kans. Käynnistä taustalle ylemmillä ohjeilla.

projektihakemistossa (missä tuo ExpressApp on) npm install mongoose
		npm install mongodb
-> bracketsissä näkyy hakemistoina mongoose ja mongodb

bin hakemistoossa voi avata konsolin myös komennolle mongo. Aputyökalu.
